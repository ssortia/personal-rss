# 004. CSS-фреймворк: Tailwind CSS v4

**Статус:** Принято

**Дата:** 2026-02-28

## Контекст

Для стилизации используется utility-first подход с shadcn/ui-компонентами. Tailwind CSS v4 вышел в начале 2025 и кардинально изменил подход к конфигурации.

## Рассмотренные варианты

**Tailwind CSS v3 (предыдущая major-версия)**
- ✅ Стабильная, огромная экосистема, все туториалы под неё
- ✅ Хорошо документирована для shadcn/ui
- ❌ Более медленная сборка (PostCSS-based с JIT)
- ❌ Конфигурация через `tailwind.config.js` — дублирование токенов между CSS и JS

**Tailwind CSS v4** ✓
- ✅ CSS-first конфигурация: все токены определяются через `@theme {}` в CSS-файле
- ✅ Значительно быстрее сборка (новый движок на Rust/Oxide)
- ✅ Нативные CSS-переменные как первый класс — токены доступны и в Tailwind, и в inline-стилях
- ✅ Убраны лишние абстракции (`content` массив, `purge`, legacy JS-конфиг)
- ❌ Требует `@tailwindcss/postcss` вместо `tailwindcss` в PostCSS
- ❌ Синтаксис в CSS изменился (`@import "tailwindcss"` вместо `@tailwind base/components/utilities`)
- ❌ Меньше туториалов и примеров под v4

## Решение

Выбран **Tailwind CSS v4**.

Поскольку шаблон ориентирован на современный стек, v4 предпочтительнее: он убирает дублирование токенов между JS-конфигом и CSS, делает дизайн-токены нативными CSS-переменными. Это упрощает кастомизацию темы в `globals.css` без необходимости разбираться в `tailwind.config.ts`.

## Последствия

### Положительные
- Дизайн-токены определяются один раз в `src/app/globals.css` через `@theme {}`
- Нативные CSS-переменные (`--color-primary` и т.д.) доступны вне Tailwind (в inline-стилях, CSS Modules)
- Быстрее сборка

### Отрицательные / компромиссы
- PostCSS-конфиг: `{ '@tailwindcss/postcss': {} }` вместо `{ tailwindcss: {} }` — легко забыть
- При миграции существующих проектов с v3 требуется замена директив в CSS и пересмотр конфигурации
- shadcn/ui пока частично адаптирован под v4 — некоторые компоненты могут требовать ручной правки

### Риски
- Экосистема ещё адаптируется к v4; плагины (например, `prettier-plugin-tailwindcss`) могут отставать
